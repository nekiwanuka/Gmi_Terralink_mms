{% trans "The system is divided into several core modules, each handling a specific business function:" %}

  • {% trans "Inventory" %}: {% trans "Manage your products, stock levels, locations, and transfers." %}
  • {% trans "Sales" %}: {% trans "Create and manage customer invoices, payments, and receipts." %}
  • {% trans "Procurement" %}: {% trans "Manage supplier purchase orders and track incoming stock." %}
  • {% trans "Mining" %}: {% trans "Record daily mining activities, including labor and equipment costs." %}
  • {% trans "Payroll" %}: {% trans "Manage employee payroll, including deductions and allowances." %}
  • {% trans "Contacts" %}: {% trans "Manage your clients and suppliers." %}
  • {% trans "Reports" %}: {% trans "Generate various reports for sales, expenses, and inventory." %}
  • {% trans "Settings" %}: {% trans "Configure your business profile, currency, and other system settings." %}

3. {% trans "Basic Workflows" %}

{% trans "This section describes the basic steps for common tasks in the system." %}

3.1. {% trans "Creating a Sale" %}

  1. {% trans "Navigate to Sales" %}: {% trans "From the sidebar, click on 'Sales' to go to the sales list page." %}
  2. {% trans "Create a New Invoice" %}: {% trans "Click on the 'New Invoice' button." %}
  3. {% trans "Select a Client" %}: {% trans "Choose an existing client or create a new one." %}
  4. {% trans "Add Items" %}: {% trans "Add items from your inventory to the invoice. Specify the quantity and price." %}
  5. {% trans "Save the Invoice" %}: {% trans "Once all the details are entered, save the invoice." %}
  6. {% trans "Record Payment" %}: {% trans "After the invoice is created, you can record a payment against it." %}

3.2. {% trans "Adding a New Inventory Item" %}

  1. {% trans "Navigate to Inventory" %}: {% trans "From the sidebar, click on 'Inventory'." %}
  2. {% trans "Create a New Item" %}: {% trans "Click on the 'New Item' button." %}
  3. {% trans "Fill in Details" %}: {% trans "Provide the item name, category, cost price, selling price, and other relevant details." %}
  4. {% trans "Save the Item" %}: {% trans "Click 'Save' to add the new item to your inventory." %}

3.3. {% trans "Recording a Mining Entry" %}

  1. {% trans "Navigate to Mining" %}: {% trans "From the sidebar, click on 'Mining'." %}
  2. {% trans "Create a New Entry" %}: {% trans "Click on 'New Mining Entry'." %}
  3. {% trans "Enter Details" %}: {% trans "Fill in the details for the mining activity, including the date, site, and costs." %}
  4. {% trans "Save the Entry" %}: {% trans "Save the entry to record the mining activity." %}
